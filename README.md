# Durak Sefer Sorgu

GitHub Pages üzerinde doğrudan çalışabilen, dış bağımlılığı olmayan tek sayfalık durak ve sefer sorgulama uygulaması.

## İçerik

- `index.html`: Uygulama ve gömülü güncel veri
- `.nojekyll`: GitHub Pages'in Jekyll işlemesini devre dışı bırakır
- `README.md`: Kurulum ve veri bilgileri

## GitHub Pages kurulumu

1. ZIP dosyasını açın ve üç dosyayı GitHub deponuzun ana dizinine yükleyin.
2. GitHub'da **Settings → Pages** bölümüne gidin.
3. **Deploy from a branch** seçeneğini, `main` dalını ve `/ (root)` klasörünü seçin.
4. Kaydettikten sonra GitHub'ın oluşturduğu Pages adresini açın.

## Eşleştirme notu

Yeni durak listesinde durak kodu sütunu bulunmadığından, durak kodları önceki HTML'deki kod–durak adı ilişkisiyle eşleştirilmiştir. Aynı güzergâh, aynı hat/yön ve durak adı öncelikli kullanılmış; küçük yazım değişikliklerinde yüksek benzerlikli eşleştirme uygulanmıştır. Kod karşılığı bulunamayan **20** satır veri setine eklenmemiştir.

Uygulama herhangi bir sunucu, API veya ek JavaScript dosyası gerektirmez.
