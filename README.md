# Durak Sefer Sorgu

Bu paket GitHub Pages üzerinde yayınlanmak üzere hazırlanmıştır.

## Kullanım

1. ZIP dosyasını çıkarın.
2. İçindeki `index.html`, `.nojekyll` ve `README.md` dosyalarını GitHub repository ana dizinine yükleyin.
3. Repository ayarlarından `Settings > Pages` bölümüne girin.
4. `Deploy from a branch` seçeneğini kullanarak `main` branch ve `/root` klasörünü seçin.
