# Durak Sefer Sorgu

Bu paket GitHub Pages üzerinde yayınlanmak üzere hazırlanmıştır.

## İçerik

- `index.html`: Güncel durak sefer sorgu uygulaması
- `.nojekyll`: GitHub Pages yardımcı dosyası
- `README.md`: Bu açıklama dosyası

## Yayına alma

ZIP'i çıkarın ve içindeki dosyaları repository ana dizinine yükleyin. GitHub Pages ayarında `main` branch ve `/root` klasörünü seçin.
