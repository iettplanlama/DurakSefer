# Durak Sefer Sorgu

Bu paket GitHub Pages'e doğrudan yüklenmek için hazırlanmıştır.

## Kullanım

1. GitHub'da yeni bir repository oluşturun.
2. Bu ZIP içindeki dosyaları repository'nin ana dizinine yükleyin.
3. `Settings > Pages` bölümünden `Deploy from a branch` seçin.
4. Branch olarak `main`, klasör olarak `/root` seçip kaydedin.

Ana uygulama dosyası `index.html` olarak adlandırılmıştır.
